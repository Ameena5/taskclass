    • GIT COMMANDS

    1. GIT INIT

    Git init is a one time command you use during the intial setup of a new repository.

    2.GIT CLONE :

    Git clone in the terminal, paste the URL you copied earlier, and press "enter" to create your local clone.


    copy the code and paste it in git clone


    3. GIT ADD :

    The git add command adds a change in the working directory select that files, and moves to the staging area.



    4.GIT STATUS :

    Git status command displays the state of the working directory and the staging area.

    5.GIT BRANCH :

    Git branch command lets you create, list, rename, and delete branches.

    6.GIT COMMIT :

    Git commit command captures a snapshot of the project's currently staged changes.

    7.GIT PUSH :

    Git push command is used to upload local repository content to a remote repository.

    8.GIT PULL :

    Git pull command is used to retrieve and download content from a remote repository and update the local repository as soon as it has been downloaded.
